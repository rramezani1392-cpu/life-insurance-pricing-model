
Life Insurance Pricing Model 📊 Project Overview 

This project presents an end-to-end actuarial model for pricing life insurance products using Python.

It includes:

Mortality modeling (q_x estimation) Monte Carlo simulation for risk estimation Expected loss calculation Premium pricing logic 🧠 Key Features Synthetic insurance dataset generation Risk modeling based on age, health, and smoking status Simulation of death events using probabilistic methods Financial modeling using present value concepts 🛠 Tools & Technologies Python (NumPy, Pandas) Power BI (for dashboard visualization) Monte Carlo Simulation 📈 Results 

The model estimates expected losses and calculates premiums by incorporating risk, time value of money, and business margins.
